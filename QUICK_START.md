## Create Your First Daily Post

```powershell
# From the project root
.\scripts\new-post.ps1
```

That's it! The script will:
1. Create a new post file with today's date
2. Fill it with a helpful template
3. Open it for you to start writing

## With a Custom Title

```powershell
.\scripts\new-post.ps1 "Learning React Hooks"
```
